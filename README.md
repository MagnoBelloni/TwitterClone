# TwitterClone

Este projeto foi construído pensando no backend do [Twitter](https://twitter.com/).

## Início

### Pré-requisitos:

Você precisa do NodeJS e do Yarn instalado em sua máquina.
Além disso crie um arquivo .env na raiz do projeto e coloque a chave como o exemplo abaixo. O exemplo também pode ser visualizado com o arquivo: [.env.example](https://github.com/MagnoBelloni/TwitterClone/blob/main/.env.example).

```
HTTP_PORT=3000
MONGO_URI=
JWT_SECRET=
```

```
yarn
yarn start
```
